## Overview
**mGBA Pokemon Emulator** is a highly optimized and accurate emulator for playing classic Pokémon games (Generation 3 and others) on Windows PC. It offers excellent performance, high compatibility, and many useful features for a comfortable gaming experience.

> [!Important]
> This emulator is for educational and game preservation purposes only. You must legally own the original game cartridges to use ROM files.

## Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| Emulator doesn't start     | Run as Administrator                          |
| Game runs slow             | Enable "Speed Up" or lower resolution         |
| Black screen               | Change video renderer (OpenGL / Direct3D)     |
| Antivirus warning          | Add the folder to exclusions                  |
| Controls not working       | Configure input in Settings → Input           |
